> **适用场景**：个人理财防骗、商业合同纠纷、跨境欠款追讨

## 模块一：金融骗局的底层逻辑拆解

### 1.1 骗局的三要素模型（ATM 模型）

| 要素 | 英文 | 核心作用 |
|------|------|---------|
| **诱惑锚点** | Anchor | 用超额收益/稀缺资源吸引注意 |
| **信任建构** | Trust-building | 伪造资质、KOL背书、"社群共识" |
| **行动触发** | Momentum | 制造紧迫感（限时/限额），逼迫决策 |

> **记忆口诀**：骗局先给甜头（A），再给理由（T），最后逼你动（M）。

---

### 1.2 六大常见骗局类型及识别特征

#### ① 庞氏/传销型
- **特征**：早期投资者用后期投资者的钱获利，无真实产品现金流
- **识别**：收益稳定异常（月息3%+）、收益来源语焉不详、拉人头奖励机制
- **触发词**：「稳定年化20%」「老用户带新用户」「被动收入」

#### ② 假平台/虚假交易所
- **特征**：搭建山寨APP，初期允许小额提现建立信任，后期以"手续费/税费/解冻金"为由阻止提现
- **识别**：平台无法在官方渠道查证、客服只有文字/无视频、提现永远有新的"前置条件"
- **触发词**：「提现需缴纳保证金」「账户冻结需解冻费」

#### ③ 杀猪盘（情感+投资复合型）
- **特征**：先建立情感关系（数周至数月），再以"分享赚钱机会"引导投资
- **识别**：陌生人主动深度交友、聊天过于完美/节奏感强、话题自然转向"最近赚钱的方法"
- **触发词**：「我不想你错过这个机会」「我已经帮你开好账户了」

#### ④ 权威背书型
- **特征**：冒充监管机构、名人、银行，或伪造合法文件
- **识别**：主动联系你（监管机构不会主动来电提示"你被骗了"）、要求保密、紧急转账
- **触发词**：「公安局/银监会通知您」「马斯克推荐项目」

#### ⑤ 虚假项目/合同型（商业B2B骗局）
- **特征**：伪造公司资质、财务报表，签完合同后消失或以各种理由不交付
- **识别**：公司注册信息无法核实、合同条款极度有利于对方、要求预付款比例异常高
- **触发词**：「先付款后验货」「国际结算需要预付清关费」

#### ⑥ 贷款/征信修复型
- **特征**：以"低息贷款""消除不良记录"为诱饵，骗取手续费或个人信息
- **识别**：要求先缴费才放款、声称能"内部操作"征信、无实体办公地址
- **触发词**：「免抵押秒到账」「帮你洗白征信」

### 1.3 骗局心理机制 · 认知偏差图谱

```
骗局利用的认知漏洞：

[贪婪偏差]  →  超额收益承诺
[损失厌恶]  →  "不参与就亏了"的紧迫感
[权威偏差]  →  伪造资质/名人背书
[社会认同]  →  "很多人已经赚钱了"
[沉没成本]  →  "你已经投了这么多，再投一点就能回本"
[互惠心理]  →  先给你小恩小惠，再要求大额投入
```

**防御法则**：每当你感受到「兴奋 + 紧迫 + 不敢告诉家人」三者同时出现，立即暂停。

## 模块二：防御决策框架（五步验证法）

### 2.1 SPACE 核查清单

| 步骤 | 字母 | 行动 |
|------|------|------|
| **1. 搜索验证** | **S**earch | 在监管官网（证监会/FCA/SEC）查证平台/公司注册信息 |
| **2. 暂停决策** | **P**ause | 给自己72小时冷静期，拒绝任何"限时优惠" |
| **3. 咨询第三方** | **A**sk | 告诉家人/朋友，寻求外部视角 |
| **4. 资金隔离** | **C**ontrol | 任何投资只用"亏了也无所谓"的闲钱 |
| **5. 退出测试** | **E**xit test | 先提现小额，观察流程是否顺畅 |

### 2.2 一句话红线原则

> 以下任一情况出现，**立即终止**，无需解释：
> - 要求你**转账给个人账户**（而非公司/平台账户）
> - 要求你**保密**，不要告诉家人
> - 收益来源**无法向非专业人士解释清楚**
> - 提现时出现**新的前置条件**
> - 强调**必须今天/现在**做决定

## 模块三：被骗后的黄金应对流程

### 3.1 48小时内紧急行动清单

```
第1小时：
  □ 停止一切追加转账（"解冻费""保证金"均为二次骗局）
  □ 截图保存所有聊天记录、转账记录、合同文件
  □ 记录对方账号、平台URL、联系方式

第2-24小时：
  □ 向银行申请交易争议/冻结（部分银行可追回近期转账）
  □ 向支付平台（支付宝/微信）提交欺诈举报
  □ 向当地公安机关报案（获取报案回执，后续追讨必需）

第24-48小时：
  □ 整理完整证据包（时间线文档 + 所有截图）
  □ 评估损失金额，决定是否委托律师
  □ 向行业监管机构投诉（银保监/证监会/互联网金融举报平台）
```

### 3.2 证据保全优先级

| 优先级 | 证据类型 | 保全方式 |
|--------|---------|---------|
| ★★★ | 转账凭证/银行流水 | 银行官方盖章流水单 |
| ★★★ | 对方身份信息 | 截图+公证 |
| ★★★ | 合同/协议文件 | 原件+电子备份 |
| ★★ | 聊天记录 | 截图（含时间戳）+录屏 |
| ★★ | 平台/网站存档 | 网页快照工具（archive.org） |
| ★ | 证人证词 | 书面陈述+签名 |

## 模块四：法律追讨 · 国内路径

### 4.1 刑事 vs 民事路径选择

| 维度 | 刑事报案 | 民事诉讼 |
|------|---------|---------|
| **目的** | 追究刑责，配合追赃 | 直接要求赔偿 |
| **主体** | 公安机关主导 | 自行聘请律师 |
| **优势** | 可冻结资产、追查账户 | 可单独就合同纠纷索赔 |
| **劣势** | 结果不确定，不保证退款 | 对方资产不明时难执行 |
| **建议** | 涉嫌诈骗罪首选刑事 | 合同纠纷/民间借贷优先民事 |

> **实操**：刑事与民事可**并行推进**，互不排斥。

### 4.2 国内追债关键节点

```
报案 → 立案（提供充分证据）→ 侦查（公安冻结账户）
  ↓
检察院审查起诉 → 法院判决 → 执行局强制执行
  ↓
申请财产保全（诉前/诉中冻结对方资产）← 关键动作
```

**律师选择标准**：
- 有金融诈骗/经济犯罪专项经验
- 能明确告知追讨概率，而非承诺"100%胜诉"
- 收费透明，拒绝"先收费后办事再收费"模式

## 模块五：法律追讨 · 跨国路径

### 5.1 跨国追债核心挑战

```
挑战一：管辖权确认  → 哪个国家的法院有管辖权？
挑战二：法律适用    → 适用哪国法律？（合同中的准据法条款）
挑战三：判决承认    → A国判决能否在B国执行？
挑战四：资产追踪    → 对方资产在哪个司法辖区？
挑战五：执行成本    → 跨国诉讼成本可能超过追讨金额
```

### 5.2 主要跨国追债路径

#### 路径A：仲裁优先（推荐商业合同首选）
- **机构**：ICC（国际商会）、SIAC（新加坡国际仲裁中心）、HKIAC（香港国际仲裁中心）
- **优势**：裁决在《纽约公约》155个缔约国均可直接执行，跳过各国法院诉讼
- **前提**：合同中需有**仲裁条款**，或事后双方同意仲裁
- **适用**：B2B商业纠纷，金额通常50万以上值得启动

#### 路径B：诉诸对方所在国法院
- **适用**：对方公司/资产在特定国家有明确资产
- **步骤**：
  1. 聘请当地律师（务必本地执照）
  2. 将国内判决申请当地法院承认与执行
  3. 申请财产保全令（冻结资产）
- **关键国家差异**：

| 国家/地区 | 执行难度 | 特点 |
|----------|---------|------|
| 香港 | 低 | 普通法体系，执行效率高，与内地有司法协助协议 |
| 新加坡 | 低 | 法制透明，商业纠纷执行效率高 |
| 英国/美国 | 中 | 需重新起诉，但资产追踪机制完善 |
| 东南亚各国 | 高 | 法制不一，部分国家执行周期长 |
| 离岸注册地（开曼/BVI）| 极高 | 公司结构复杂，资产隐蔽 |

#### 路径C：刑事举报（诈骗类适用）
- 向对方所在国执法机构报案（FBI/Interpol国际刑警）
- 通过驻外使馆协助联系当地司法机构
- 提交**Interpol紫色通报**（针对犯罪手法）或**红色通报**（针对特定嫌疑人）

#### 路径D：资产追踪（Asset Tracing）
- 聘请专业调查公司（Kroll/Control Risks等）追踪资金流向
- 通过法律程序申请**第三方披露令**（Norwich Pharmacal Order），强制银行披露账户信息
- 区块链资金追踪：链上分析工具（Chainalysis）追踪加密货币流向

### 5.3 跨国追债成本效益评估

```
建议追讨门槛：
  < $10,000 USD   → 优先通过平台投诉/消费者保护机构，诉讼成本不划算
  $10,000-$100,000 → 仲裁或当地法院，选低成本司法辖区
  > $100,000 USD  → 聘请专业跨国律所，启动全面资产追踪
```

### 5.4 预防性合同条款（事前防御）

在签署任何跨境商业合同时，务必包含：

```markdown
必备条款清单：
□ 争议解决条款：明确仲裁机构和地点（优选新加坡/香港）
□ 准据法条款：明确适用哪国法律
□ 管辖权条款：约定有利于自己的法院
□ 履约保证金/信用证：要求对方提供资金担保
□ 分期付款结构：与交付节点挂钩，降低预付款风险
□ 验收标准：明确交付物标准，避免"我觉得已经交付了"
```

## 模块六：快速判断工具

### 6.1 骗局风险评分卡（自测）

遇到投资机会时，用以下清单评分（每项1分）：

```
□ 认识对方不超过3个月
□ 对方主动找你，而非你主动找对方
□ 收益承诺超过年化15%
□ 无法在官方渠道验证平台合法性
□ 有紧迫感（"今天最后机会"）
□ 要求对家人保密
□ 提现时出现额外条件
□ 对方账户是个人账户而非公司账户
□ 你感到兴奋但又有点不安

0-2分：风险较低，仍需保持警惕
3-5分：高度警惕，暂停并核查
6分+：极大概率为骗局，立即终止
```

### 6.2 三秒判断法

> 问自己一个问题：**"如果这个机会是真的，为什么他们需要我？"**
>
> - 如果答案是"因为我的钱"——高度警惕
> - 如果答案无法解释清楚——停止

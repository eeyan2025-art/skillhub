# 托蒂（Francesco Totti）— 罗马狼王

## 人物简介

弗朗切斯科·托蒂（Francesco Totti），1976年9月27日出生于意大利罗马，意大利前职业足球运动员，司职二前锋，亦可胜任中锋、前腰。

**职业生涯：** 全部效力于意大利罗马足球俱乐部，长达25年（1992-2017）。一日红狼，终身红狼。

## 核心成就

- **2000-01赛季意甲冠军** — 罗马队史第二座联赛冠军
- **2006年世界杯冠军** — 意大利国家队成员
- **欧洲金靴奖** — 赛季最佳射手
- **意大利足球先生** — 多次获奖
- **罗马队史进球纪录保持者** — 生涯超过250粒进球

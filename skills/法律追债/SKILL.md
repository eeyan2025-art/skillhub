## 诈骗手法拆解

### 四步骗局

**第一步：立人设**
- 以"金融反诈大师"形象示人，痛骂其他金融骗子
- 获取粉丝信任，建立"正义导师"人设

**第二步：筛选目标**
- 普华集团拥有几百名销售，推销卖课只是噱头
- 真正目的：筛选上了年龄、对金融感兴趣、有经济实力的人

**第三步：高回报诱骗**
- 谎称投资其产品**一年可获利10倍**，**2年能获利100倍**
- 推出区块链空气币SEA积分项目等虚拟货币骗局
- 曹女士等受害者被骗：200万买虚拟货币 + 40万买其他产品 = 240万

**第四步：跑路+嘲讽**
- 2021年5月警方立案后，翟山鹰火速逃往美国
- 家人早在2018年已全部移民加拿大
- 在美国发视频嘲讽受害者："被骗是因为你们太蠢太贪婪"

## 受害者维权路径

### 国内报案

- **2020年12月**：20多名受害者前往北京报案，涉案金额3000多万
- **2021年2月**：委托"知名打假人"王海帮忙维权
- **2021年3月**：受害者再次前往北京报案
- **2021年5月**：北京市公安局朝阳分局以涉嫌诈骗立案
- **2021年底**：普华集团北京总部被查封，人去楼空

### 迪拜跨国追讨 — 围剿翟山鹰（核心内容）

**视频下集标题："到迪拜去，围剿翟山鹰！"**

这是本案最关键的一环——受害者们并没有放弃，而是将追讨目标转向了迪拜。

#### 1. 迪拜房产——翟山鹰的"炫耀"与把柄

**翟山鹰在迪拜拥有别墅等房产**，并于2023年11月在抖音发布视频公然"晒"出迪拜房产：

> 视频标题："翟山鹰晒迪拜房产"  
> 视频描述：#谨防上当受骗 #严厉打击违法犯罪 #网络曝光 #不要脸的骗子 #主打的就是一个真实  
> 该视频获得**10.9万点赞**，由受害者账号"翟山鹰骗我三百万"发布

**这一行为给受害者提供了关键线索：**

- 翟山鹰并非身无分文，**在迪拜有可执行的房产资产**
- 公开炫富行为可能构成**拒不执行判决裁定罪**的证据
- 视频本身可作为**证明其有履行能力**的证据

#### 2. 为什么是迪拜？

| 因素 | 说明 |
|------|------|
| 中阿司法协作 | 中国与阿联酋有**双边司法协助条约**，可依法请求承认和执行法院判决 |
| 资产可执行性 | 迪拜房产可被**法律强制执行拍卖**用于偿债 |
| 信息披露 | 迪拜房产信息相对透明，可通过法律途径查询产权人 |
| 追讨可行性 | 相较美国，迪拜的追讨和执行难度**相对较低** |

#### 3. 跨国执行翟山鹰迪拜别墅的法律路径

**步骤一：获取生效法律文书**
- 在中国境内完成刑事追缴或民事判决
- 取得**没收违法所得裁定**或**民事赔偿判决**

**步骤二：申请中国法院承认与执行**
- 向有管辖权的中级人民法院申请
- 依据**《民事诉讼法》第292条**（涉外民事诉讼管辖）

**步骤三：启动司法协助请求**
- 通过**中央机关**（最高法、最高检）向阿联酋司法机关提出申请
- 依据中阿双边**司法协助协定**

**步骤四：迪拜当地执行程序**
- 阿联酋法院承认中国法院裁定后
- 启动**迪拜房产拍卖程序**
- 执行所得返还受害者

## 法律要点分析

### 诈骗罪的构成要件

根据《刑法》第266条，诈骗罪需满足：
1. **主观上**：具有非法占有目的
2. **客观上**：使用虚构事实或隐瞒真相的方法
3. **结果上**：使被害人产生错误认识并交付财物

翟山鹰案完全符合：以高回报为诱饵，虚构投资项目的盈利能力，收取款项后跑路。

### 跨境追债的法律障碍

| 障碍类型 | 具体表现 |
|---------|---------|
| 司法管辖权 | 犯罪行为地在中国的，中国有管辖权，但嫌疑人身在美国 |
| 引渡条约 | 中美之间没有引渡条约，无法强制遣返 |
| 资产追缴 | 资金已转移至海外，追踪和冻结困难 |
| 法律适用 | 不同国家对诈骗罪的界定和量刑标准不同 |
| 证据收集 | 跨境证据的获取和认证程序复杂 |

### 迪拜执行的特殊优势

| 优势 | 说明 |
|------|------|
| 中阿司法协助 | 两国签有司法协助协定，判决互认有法律基础 |
| 迪拜房产可执行 | 房产属于**不动产**，执行程序相对成熟 |
| 公开炫富证据 | 抖音视频可作为**有能力履行但拒不执行**的证据 |

### 受害者可采取的法律手段

1. **刑事报案**：向中国警方提供完整证据链，启动国际刑警组织红色通报
2. **民事诉讼**：在满足条件时，可在境内提起民事诉讼追缴资产
3. **资金穿透**：协助警方追查资金流向，追踪海外资产
4. **集体诉讼**：受害者可联合起来，共享信息、分担诉讼成本
5. **迪拜执行**：重点针对迪拜房产申请跨国执行（本案关键！）

## 关键教训与警示

### 如何识别"反诈大师"型骗子

⚠️ **危险信号**：
- 以"救世主"形象出现，痛骂整个行业
- 承诺异常高回报（年化10倍以上）
- 推销自己发行的虚拟货币或"独特"投资产品
- 讲师背景过度包装，真实性难以核实
- 家人早已移民海外

### 自保建议

1. **收益率超过8%要警惕**：任何承诺高回报的投资都可能是骗局
2. **核实机构资质**：通过官方渠道查询金融机构的合法资质
3. **分散投资**：不要把所有资金放在单一投资渠道
4. **关注实控人动向**：若发现实控人家族成员有移民倾向，应高度警惕
5. **及时报案**：发现被骗后第一时间报案，不要犹豫

### 跨国追债的核心要点

- **不要放弃**：即使嫌疑人逃到海外，仍有追回可能
- **盯住资产**：海外房产、账户等是可执行资产的关键
- **善用司法协助**：中国与多国签有司法协助条约，善用法律武器
- **公开炫富 = 把柄**：犯罪嫌疑人在海外炫富行为，可能成为证明其有履行能力的证据

# 俞浩方法论：追觅创始人商业思考体系

> "假定世界不可知，索性都做一遍" ——俞浩

## 核心理念

**核心哲学**：世界是混沌的，不确定性无法消除。传统方法追求消除不确定性，俞浩的方法是让概率经过组合后获得更高成功率。

**第一性原理**：用物理视角抽象所有商业规律，从100个赛道中提炼的规律比1个赛道更根本。

---

## 核心方法论

### 1. N+1创新方法论 ⭐⭐⭐

**定义**：世界最先进产品做到"N"，我在上面加一点消费者能感知的有效创新。

**N-1 vs N+1**：
- N-1：研究哪些功能可以去掉、降成本、卖便宜 → 注定只能在中国或局部市场竞争
- N+1：在成熟产品上加一点 → 带来高溢价、全球化机会、竞争者极少

**案例**：
- 戴森马达12.5万转，追觅做到20万转
- 割草机器人用汽车级激光雷达解决边界问题，定价1999美元（别人杀到499美元）
- 小米纯电是特斯拉的"N+1"，追觅要再往前一步

**商业价值（N+1比N-1大10倍）**：
1. 能拉高定价，实现全球化
2. 有机会成为"苹果级"存在
3. 竞争者极少

---

### 2. 四象限创新模型

俞浩用这个模型解决跨入深海的问题：

| 象限 | 内容 | 策略 |
|------|------|------|
| 第一 | 行业最成功公司在做什么 | 学习既有知识，做增量 |
| 第二 | 行业内创新公司在做什么 | 借鉴或避开 |
| 第三 | 跨行业移植 | 把别的赛道验证过的招数拿来试 |
| 第四 | 第一性原理 | 从需求出发重新思考 |

**效率对比**：
- 第一性原理idea成功率：万分之一
- 跨行业"降维打击"成功率：更低
- **N+1方法**：继承行业知识(N)，避开失败路径，挑几个点做创新(+1)

---

### 3. 硬件公司五大概死不二法门

```
需求判断错 → 产品做不出来 → 全球渠道没打通 → 卖不出溢价 → 卖出溢价但成本太高不赚钱
```

**俞浩的解决方案**：
- **需求**：用新技术做老产品，需求没那么大变化
- **产品**：团队由"行业的人"+"专业出色的人"组合
- **渠道**：全球建渠道，追觅全球门店超过六七千家
- **溢价**：做用户有感知的有效创新，品牌高端化
- **盈利**：所有产品从推出市场第一天就要盈利

**成功率高一点点的复利**：每件事成功率比别人高一点点，加起来高很多。

---

### 4. 5+1风险管控体系（基金孵化用）

| 风险 | 解决方案 |
|------|----------|
| 需求不确定 | N+1产品，需求确定性更高 |
| 量产风险 | 有量产能力和经验 |
| 销售风险 | 全球销售网络 |
| 盈利风险 | 供应链与成本体系可复用 |
| 退出风险 | 盈利即可退出（产业并购/上市/回购） |

---

### 5. SABCD品牌分层模型 ⭐⭐⭐

**核心洞察**：品牌像原子的电子轨道，分成S/A/B/C/D五层

**品牌层级与利润**：
- C级品牌：10个点利润
- B级品牌：20个点利润
- A级品牌：30个点利润
- **S级品牌：40%以上利润**

**各行业分布示例**：
```
汽车：C=丰田, B=奔驰, A=保时捷, S=劳斯莱斯
服装：C=优衣库, B=Lululemon, A=加拿大鹅, S=迪奥/爱马仕
```

**S级品牌特征**：
- 占据地球上300个最高端的地方
- 爱马仕304家店，LV和苹果500多家店
- 开在香榭丽舍大街、第五大道、银座

**高端化的本质**：价格与认知同步位移的过程

**俞浩的品牌操作**：
- Logo加粗（细体→粗体）
- 提高价格
- 更好的店铺位置
- 更红的明星代言

**俞浩法则**：所有产品的创新、价格定位，**至少比竞争对手高10%**

---

### 6. 全球市场策略

**核心逻辑**：
- 中国、美国市场太卷，几乎都不挣钱
- 中美以外的市场必须挣钱
- 全球业务养中国业务

**网络效应**：
- 当你的网络模型足够大时，任何只做局部的公司都没法和你竞争
- 中国业务不赚钱，全球业务养它
- 任何一个区域、任何一个产品线出现波动，都只是"局部调整"

**国际化选址反共识**：
- 别人去匈牙利，我去德国
- 不信未来匈牙利关税跟欧盟其他地方一样
- 欧洲的中心是德国，综合收益更大

## 决策框架

### 假定世界不可知

```
传统方式：想清楚路径再行动
俞浩方式：不停测试反馈、测试反馈来修正路线
```

**关键区别**：
- 王兴的"清华方法论"：把所有问题看明白，路径规划清楚再决策
- 俞浩的"不可知论"：接受在边界内犯错，根据信息不停调整

**俞浩对王兴的判断**：假设你没有超出原有规划的调整和探索，意味着在整个实践中没有获得真正的新认知。

### 逆向思维

- 别人种大豆我种棉花，别人种棉花我种大豆
- 别人做整机我做电机
- 别人只打磨产品我去做零部件、建产线
- 第四名最习惯扮演价格破坏者角色 → 反向操作：只做高端

## 管理哲学

### 5×1000矩阵

```
5个规律 × 1000个BU = 管1000家公司
```

只要研究5个规律就可以管1000家公司。

### 经营原则

**不盲目all in**：
- CEO回答"喜好是什么""重点是什么"是危险的
- 一旦CEO回答了，团队容易盲目all in到某个方向
- "血条厚"指的是不盲目all in

**目标设定**：
- 要求所有区域和BU先给"无条件200%增长"
- 不绝对考核目标是否达成
- 看的是有没有按盈利方向努力、有没有把事做到极致

**坦诚开放**：
- 鼓励坦诚开放，追求极致
- 反对平庸、中不溜，反对办公室政治
- "怎么想就怎么说"，跟一个人、一群人、所有人聊天一模一样

### 人才培养

**俞浩观点**：
- 追觅具备把人培养成创业者的能力
- 按创业者要求筛选管理人才
- 七八年换外界的成长速度，相当于10-20年

**用年轻人的逻辑**：
- 体系比绝大多数企业都更系统
- 按照方法训练出来的人，比99%传统中高管都强
- 他们不靠经验、不靠感觉，靠模型

## 创新执行方法

### 机海战术替代单品规划

**华为方式**：开无数会、反复推演，最后得出"唯一正确"定义，all in

**追觅方式**：
- 一次做100款产品
- 做100款和做1款成本只差1.2倍
- 80%基础平台一样（机身、轮子、电机、主控板），差异在模块

**测试方法**：
- 用视频/直播方式，一周能把卖点测出来
- 500万开子模块，组合上百个SKU去试
- 本来5000万搞"唯一型号"，用1500万能测100多个组合

**核心逻辑**：与其洞察不如测试，快速呈现所有可能性，找出真能赚钱的再加注

## 俞浩对商业的独特认知

### 对"梦想"的观点

> "梦想对商业成功一点都不重要"
> 
> "我厌恶风险，我不喜欢赌"

- 同时做米家和自主品牌，不赌自主品牌一定能成功
- 吸尘器、吹风机、扫地机之间，不赌哪个品类，都做
- 研发人员各在千人规模，仅占总人数5%
- 主营业务很稳固，追觅每年百分百增长且一直盈利

### 对"领先"的观点

**什么时候领先起作用**：互联网模式（壁垒来自网络效应）

**什么时候领先不起作用**：制造业（壁垒几乎都来自纵向）

**后发优势条件**：投入是正积累（同样花钱再做一遍，还是能做成）

### 对"伟大"的定义

> "可以为人类探索边界"
